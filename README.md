## GRAPE API DEMO  

### Set Up
Clone the repo  
`git clone https://github.com/juandefelix/grape_api && cd grape_api`  
Get the neccessary gems:  
`bundle install`  
Start the server:  
`rackup config.ru`  
You can open in your browser the file `index.html` and see a simple display of the results of calling the API methods.
